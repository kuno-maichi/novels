　というわけで、公開交換日記をつけることになったのだけど。
　そういえばなんで｜双葉鈴《ふたばすず》の事を好きになったのだろうか。
　ちょうどいいから、これを最初の日記ネタにしてみようか。

$ cd exchange_diary
$ gvim diary.md

　ちなみに、vimあるいはgvimは一部熱烈な愛好家を誇るエディターだ。
　hjklで上下左右移動が出来たり、ホームポジションから指をあまり離さずに済む
　そんな操作性が人気があったりする。

―――――――――――――――――――――――――――――
　りん、告白してくれてありがとうな。
　俺も実は凄い嬉しい。
　これから恋人として色々やっていければと思っている。

　ところで、俺はなんでりんの事を好きになったんだろうなと考えてた。
　「え？先輩は覚えてないんですか？」とかツッコまれそうだけど。

　考えてみると、元々、お前とは団地でのお向かいさんなわけで。
　りんはよく家に遊びに来てたよな。
　何故か母さんとお前がお菓子作ってた光景があるのが妙だけど。

「先輩、美味しいですか？」

　てニコニコしてお菓子の感想を聞いてきたのをよく覚えてるよ。
　たぶん、小学校低学年の頃だけど、この頃からお世話好きだったんだな。
　俺の部屋が散らかってると何故か片付けようとして来たし。

　小学校はどうだっけと思い返していたんだけど、やたらこっちの
　クラスに遊びに来てたのを思い出した。
　小学校は一学年一クラで確か20人くらいだから、印象に残ってる。

「なんで、俺のとこに来るんだ？同い歳の方がいいんじゃないのか？」

　そんな感じの質問をしたような気がしてるのだけど、

「私のクラス、あんまりおもしろくないですから」

　なんて返事をしてたっけ。
　なんていうか、りんは昔からマセてたからな。
　一学年上の方が肌にあってたのかもな。

　休み時間も昼休みも俺たちのとこに入り浸るもんだから、
　すっかり可愛がられてたよな。
　この頃は単なる妹分って感じだった気がする。

　小六の頃になって、中学受験をしなくちゃいけなくなった時。
　受験勉強はしんどくなかったけど、俺だけ別の中高に行くのが寂しかった記憶がある。家から電車で三十分の距離とはいえ、りんともあまり遊べなくなるんだろうな、ってそんな事を考えたら、悲しくなったっけ。

　そういえば、その頃に、りんは

「私も先輩と同じ中学に行ってみたいです！」

　って言ってた記憶があるんだけど、あれはどういう意味だったんだろう。
　単にたまたま志望校が同じだっただけ？
　それとも……これはさすがに自惚れだよな。
　理由、返事で聞かせてくれると助かる。

　ともあれ、俺が中二になって、りんと同じ中学に通えるとなった時はそりゃあもう嬉しかったのを覚えてる。あの頃のお前はまだちんちくりんだったけど、昔からの仲間と一緒に過ごせるのがなんだか嬉しくてな。あの時は照れくさくて素直に言えなかったけど、せっかくならこの場で書いておこうと思う。

　ここまで書いて、そういえば、俺が中三の頃に少しりんの事を意識し始めたような気がする。部活も家庭科部なんて女子率が高いところに入ってしまったけど、なんていうかお前はその頃にはだいぶ可愛い感じに成長してたから、いつかワンチャンあるんじゃないか、みたいなそんな事を少し思ってた。

　書いてて、全然動機がピュアじゃないなと気づいてしまったんだけど、りんから料理について色々教わるのも楽しかったし、逆に、俺が練習してりんをぎゃふんと言わせた事も楽しかった。

　文化祭もなんとなく気軽なのもあって、シフトじゃない時は一緒に遊び歩いたよな。りんは何でも元気一杯だったから、毎年、俺の方が引っ張られてたな。今年の文化祭は恋人としてきっと楽しめるんだろうなと思うと、今からワクワクしてる。

　あ、それと今思い出したんだけど、人間関係の悩み相談にもよく乗ってもらったっけ。俺は妙に正義感が強いていうか潔癖症なところがあるから、揉め事やいじめがあると首を突っ込んで、うざがられたりした事もあって、「俺は黙ってた方がいいのかもしれないな」とかりんに聞いてもらった事も何度かあったはず。

「先輩は間違ってないです。不器用だけど、庇ってもらえて嬉しかった子もきっと多いはずです。だから、自信を持ってください」

　そんな風にして慰めてくれた事があった。りんの方が歳は下だけど、どこか年上のような落ち着いたところもあって、そんなところにも惹かれていたのかもしれないな。

　高二に上がってから、eスポーツにお互い興味を持って、ゲーミングPCを買ってもらったよな。りんが、「先輩の家にPC置かせてください！」と言って来た時はびっくりしたもんだけど、一緒に居られる時間が増えて嬉しかったのは本音だ。

　そんな感じで、りんと恋人になれたらと思うようになってた、のかもな。自分のことだけど、いつからとはっきり言えないけど。

　りんはりんの視点があって、俺とは違う思い出があるだろうし、きっかけとか思い出があったら教えてもらえると嬉しい。
―――――――――――――――――――――――――――――

$ git add diary.md
$ git commit -m "思い出話を書いてみた。文字だと色々書けてしまうもんだな。"
$ git push origin master

　交換日記を書き終えて、GitHubリポジトリにpushする。
　時間を見ると、もう午後九時。
　りんの奴は今何を考えているんだろうか。
　
$ git fetch
remote: Enumerating objects: 1, done.
remote: Counting objects: 100% (1/1), done.
remote: Compressing objects: 100% (1/1), done.

　気がついたら、りんから返事がないかなと思って、コマンドを打ち込んでいた。
　たった30分後だし、まだ返事が来るわけがないと思っていたのだけど。
　メッセージを見る限り、りんも返事をコミットしていたらしい。
　果てさて。
