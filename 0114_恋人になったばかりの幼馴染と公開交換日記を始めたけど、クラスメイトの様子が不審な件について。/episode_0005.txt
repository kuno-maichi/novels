　午後九時過ぎ。先輩と恋人になれたその日の夜。
　お風呂に入った私は、家で先輩からの手紙を待ちわびていた。

「でも、やっぱりとんでもないこと提案しちゃったかも」

　私が最初に送ったメッセージをcatしてみて恥ずかしくなってしまう。
　ちなみに、catはファイル名を指定してそれを表示するコマンドだ。

$ cat diary.md

　とすると、

――――――――――――――――――――――――――――――――
今日は先輩に告白して、受け入れてもらえた。
とても幸せ。しかも、なんと交換日記も始めることになった！
GitHubでなんて変な提案だけど、これから色々楽しみ。
改めて恋人としてよろしくお願いします。
大好きです、先輩。<- どのくらい好きなんだ？
世界一大好きです、先輩。
――――――――――――――――――――――――――――――――

　なんて表示される。
　やっぱり、恥ずかしすぎる気がしてきた。
　止めようと提案してみようか。でも、私から提案しちゃったし。
　なんで懊悩していると、GitHubに通知が。
　どうやら、先輩が日記をリポジトリexchange_diaryにpushしたらしい。

　ちなみに、GitHubはWindowsで知られているMicrosoft社が運営している。
　だから、手元で追加したファイルを送るには別の操作が必要で、それを
　pushと言う。

$ git pull origin master
remote: Enumerating objects: 1, done.
remote: Counting objects: 100% (1/1), done.
remote: Compressing objects: 100% (1/1), done.

　ちゃんと先輩が追記した日記が送られて来たようだ。
　次はコミットログを見てみないと。
　ファイルに修正をした後、修正を確定させる操作をコミットと言う。
　同時に、コミットをする時、どんな内容の修正をしたか、概要を記入出来る。
　これをコミットログと言って、複数でお話を書いたり、プログラムを書いたりする時に、他の人が変更内容を知るために重要な手がかりになる。
　コミットログを見るには、git logというコマンドを実行すればいい。

―――――――――――――――――――――――――――――
$ git log
commit c52cbfc9d40c5fe4e63eccedf2e5ae1bd839d650 (HEAD -> master)
Author: Junji Yamashita <jyamashita@qmail.com>
Date:   Wed Jun 30 21:00:30 2021 +0900

　思い出話を書いてみた。文字だと色々書けてしまうもんだな。
...
―――――――――――――――――――――――――――――

　Wedは｜Wednesday《水曜日》、Junは｜June《六月》。
　その後の30と合わせて六月三十日水曜日に書かれた事を表している。
　21:00:30は、21時00分30秒、つまり午後九時に書かれた事を表している。
　
「思い出話、かあ」

　先輩は一体何を書いたのかな？
　
$ cat diary.md

　してみた所、小学校の頃の事から、中学高校の事まで、私の事をどう思っていたかが細かく書かれていて、顔が熱くなるし、身体も火照ってくる気がする。

（文字で書かれると、こんなに恥ずかしいなんて）

　こんなに私の事を見ていてくれたなんてとっても嬉しい。
　私を意識し始めたのが先輩が中三の頃というのは、私よりも少し遅いなあ、なんて思ったり。

　でも、そっか。私が先輩と同じ中学を受けると言ったことの真意ははっきり言ったことがなかったっけ。

　私も返事を書いてみようかな。

$ emacs diary.md

　Emacsは昔はVimと人気を二分していたらしいエディタ。
　最近はVisual Studio Codeに押され気味だけど、根強い愛好者がいるらしい。
　先輩はVim派で私はEmacs派。

―――――――――――――――――――――――――――――
　先輩、思い出話をありがとうございます。
　恥ずかしいですけど、そんなに昔から私の事を見ていてくれて、とても嬉しくなっています。
　お返しに私の思い出話もお話しますね。

　私にとっても先輩はお向かいさんでしたからいつも遊びに行くのが楽しみでした。
　ちっちゃい頃は一年差でも大きかったですから、お兄ちゃんみたいな気分だったような気がします。
　私は先輩の家で将棋を打ったりオセロをしたり、人生ゲームをしたり。
　そんな事が印象に残っています。
　同じクラスの子でそういうのが好きな子は居なかったからいつも相手してくれたのはとっても嬉しかったです。

　それと、先輩たちのクラス、つまり一学年上に行ってた理由なんですけど、同じクラスの子はちょっと子どもっぽい気がしてなんだか少し退屈だったんです。マセていたと言われるとそうなのかもしれません。

　だけど、先輩も先輩の友達も私の事を妹のように可愛がってくれていつも楽しかったです。先生には「たまには同じクラスの子と遊びなさい」なんて叱られた事もありますけど、そんな事もいい思い出です。

　先輩が私と別の中高に行くのが寂しく思ってくれてたのは初めて知ったんですけど私も同じ気持ちでした。別の中高に行くと先輩と遊べる時間も減っちゃうんじゃないかなって。

　だから、私も決めたんです。先輩と同じ中学に行きたいって。私の家は先輩のところと違って中学受験は考えてなかったみたいなので、戸惑っていましたけど、お父さんお母さんにはもっと高いレベルのところで勉強したいとかなんとか言って誤魔化しました。先輩と一緒のところに行きたいなんて言ったらきっと渋い顔をされたと思いますから。

　先輩が思っていることは自惚れじゃなくて実はその通りだったんです。でも、その頃にそう意識されてたなんてちょっと恥ずかしいですね。

　今の中高は一応偏差値60くらいはあったから受かるかどうか、試験当日も合格発表の時もドキドキでした。だから、合格発表の時に私の番号があった時は飛び上がりそうに喜びました。先輩の思い出が私と一緒に通い始めた時で、合格発表の日じゃないのはちょっとおもしろいですね。

　私が先輩の事を意識し始めたのは、たぶんですけど、先輩が別の中学に行っちゃうと思った時でした。なんでこんなに寂しいのかなと考えていて、「ああ、私は先輩の事が好きなんだ」って気づいたのを覚えています。

　先輩の動機がピュアじゃないのなんて全然気にしてないです。今は想いに応えてくれたことがとても嬉しくて、ただそれだけで気持ちが一杯です。

　中学に入ってから文化祭は毎年楽しみでした。ついはしゃいじゃって先輩を振り回してしまいましたけど、笑って付き合ってくれたのはいつも感謝しています。私も今年の文化祭はとっても楽しみです。

　先輩が思っている程お役に立てたかは正直わからなかったんですけど、でも、助けになれていたなら良かったです。私は揉め事にはあんまり関わらない方だったので、先輩みたいに積極的に弱い人の味方になれる人は尊敬していましたし、だから少しでも応援したかったです。私の事を年上みたいなんて少しむず痒いです。

　実はeスポーツも最初は先輩がはじめてみるというのを聞いて、一緒に楽しみたいと思ったんです。お父さんに何やら言い訳をつけてねだって買ってもらったというのが本当のところです。今はこうして文章を書いたりプログラミングにも使っているわけですし、お父さんには感謝です。

　それからは、いつ告白しようかなってずっと考えていました。正面切って言うのは恥ずかしくて無理そうでしたし、最近よくやるようになっていたマッサージの時に、というのはその時に思いついたアイデアでした。

　実は仲良くなった整体師の先生にも、その事を少し話したことがあって、色々相談に乗ってもらいました。

　他にも思い出はいっぱいあるんですけど、とにかく今は幸せです。改めて明日からよろしくお願いします。
―――――――――――――――――――――――――――――

　書き終えて「私はなんて恥ずかしいことを書いてるんだろう」と思ったけど。
　でもそんなのは今更かもしれない。

（よし、コミットしよう）

$ git add diary.md
$ git commit 

　返事書いてみました。
　凄く色々書いてもらえて嬉しいですけど、恥ずかしいです。

　git commitはコミットするコマンドで、-mを付けると一行メッセージを書けるけど長いメッセージを書く時は-mを付けないとエディタが起動するのでこんな風にメッセージを書く。

$ git push forked master
$ git push origin master

　コミットが衝突した時に面倒なことにならないように、forkしたリポジトリにも
　pushしておく。
　これで先輩にはメッセージが届いたはず。
　でも、キリがないから今日はこの辺にしようかな。

（明日からどんな日々が待っているのだろう）

　布団に寝っ転がって明日からの日々を色々妄想する。
　手を繋いで登校したり放課後デートしたり。
　したいことは山程ある。

　そんな事を考えているとだんだんうとうとしてきた。
　（おやすみなさい先輩）
　なんて考えながら意識を手放したのだった。

「公開でこんな事やってるのアレだけど……男の方は潤二だよな？」

「試しに明日、ちょっとカマをかけてみるか」
