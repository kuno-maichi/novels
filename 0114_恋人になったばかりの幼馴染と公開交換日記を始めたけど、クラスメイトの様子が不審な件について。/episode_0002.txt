「それなんですけど。｜GitHub《ぎっとはぶ》でやりませんか？」
「え」

　やっぱり、そういう反応が来るよねー。
　わかってた。でも、一応私なりの理由があるんだ。

「理由聞かせてもらってもいいか？」

　一瞬驚いた先輩だったけど、優しく理由を聞いてくれる。
　こういう風に突拍子もないことでも耳を傾けてくれるのが好きな所の一つ。
　
「たとえば……OneDriveで交換日記をしたとして」

　ちょいちょいと私のデスクチェアに座って先輩を手招きする。
　私のゲーミングPCは先輩の家に置かせてもらっているのだ。

「うんうん。何かまずいことでもあるのか？」
「まずくはないんですけど。こう書きますよね」

『先輩、好きです』

「お前、凄い恥ずかしいこと書いてるな」
「言わないでくださいよ。例ですよ、例」

　とはいいつつ文字なら書けてしまうのが不思議なところ。

「それで、こう書き直したとしますよね」

『先輩、大好きです』

「お前、さらに恥ずかしいこと書いてるぞ」
「本音だからいいんです！とにかく、どっちかしか記録に残らないですよね」
「言われてみればそうだな。途中でりんが書き換えても俺も忘れてるかも」
「そこでGitHubですよ。｜Version《バージョン》 ｜Control《管理》｜ System《システム》
なら両方残ります」

$ mkdir exchange_diary
$ cd exchange_diary
$ git init
$ touch diary.md
$ git add .
$ git commit -m "交換日記、はじめました"

　途中まで操作をしてみる。

「りん、めちゃくちゃ操作早いけど最初からそのつもりだったな？」

　やっぱり先輩には隠し事は出来ないらしい。

「ま、いいさ。そういうのも楽しそうだしな。やろうか」

　二つ返事で受けてくれた先輩が優しくて、やっぱり大好きだと感じる。

「はい。とにかく、たとえばこんな感じで日記つけたいんですよ」

$ cat >> diary.md
# ジュン先輩と私の交換日記
# 6月30日:晴れ

今日は先輩に告白して、受け入れてもらえた。
とても幸せ。しかも、なんと交換日記も始めることになった！
GitHubでなんて変な提案だけど、これから色々楽しみ。
改めて恋人としてよろしくお願いします。
大好きです、先輩。
$ git add diary.md
$ git commit -m "交換日記1日目です！"

「なんか、目の前で見せられると凄まじく恥ずかしいな」
「私だって凄く恥ずかしいですよ。でも、例がないとやりかたわかりませんし」
「わかった。それで、訂正したい時はコミット追加すればいいんだな？」
「はい。git resetとかgit rebaseは基本的にナシでお願いします」

　だって、後で訂正したくなっても、その前のを残しておきたいのだから。
　git reset --hard "HEAD~1" とかされてgit push -fされたら悲しい。

「しかし、交換日記というか何か別のことをしてる気がしてくるな」
「Web 2.0世代の交換日記って感じがしませんか？」
「いやその……まあいいや。お互い思い出を書き溜めて行こうな」

　何か諦められてしまったけど、許してもらえるならなんでもいいや。

「ところでさ、コミットで質問追加してもいいか？こんな風に」

『大好きです、先輩。<- どのくらい好きなんだ？』

　エディタを操作した先輩が何やら書き加えている。
　これ、私以上に恥ずかしいこと書いてる気がするよー。

$ git add diary.md
$ git commit -m "質問追加してみた。出来れば答え聞かせて欲しい"

　顔がかあっと熱くなるのを感じる。
　どれくらいってそれは、一番好きだし、何なら世界一好きだ。
　でも、そんな事まで日記に書き残していいのかな。

$ git remote add origin git@github.com:yjunji@exchange_diary.git
$ git push -u origin master

　と思っているとGitHubに交換日記がpushされてしまった。
　なんだかんだ先輩もノリノリらしい。
　しかもいつの間にかGitHub上にリポジトリまで作られている。

『大好きです、先輩。<- どのくらい好きなんだ？』
『世界一大好きです、先輩』

$ git add diary.md
$ git commit -m "なんでこんな恥ずかしいこと書いてるんでしょうね。"

　交互にPCを操作してこんな事を書いてるなんて。
　きっと、世界広しと言えども私達くらいじゃないだろうか。

「こう、凄いくすぐったいな。diffも取ってみるか」

$ git diff "HEAD~1"

「差分が表示されるとなんか妙な気持ちになりますね……」

　面白半分で提案してみたのだけど、とんでもなく恥ずかしい気がしてきた。

「でも、こうして気持ち残しておけるのいい気がするな」
「はい！でも、恥ずかしすぎる愛のメッセージとかやめてくださいね？」
「それはまあ。でも、りんもだぞ」
「た、たしかに。あと、おいおい好きになった理由とかも書いていきますね」
「いいな、それ。俺もそれやるから、お互い読み合おうぜ」

　なんだか嬉しくて恥ずかしくて、でも、とても楽しいこれからが始まりそう。

　こうして、私達の、恋人同士としての日々と。
　そして、この提案によって巻き起こるドタバタが始まったのだった。
　でも、色々想いを綴れるのは本当に楽しいかもしれない。

「あ、ところで、これprivateリポジトリになってますけど」

　privateリポジトリは、招待された人しか見られない。
　つまり、今のところは私と先輩の二人だけが見られる。

「交換日記だし、そういうものじゃないか？」

　何を当然のことをという顔の先輩。

「実はその。せっかくですしpublicリポジトリにしません？」

　私の正気を疑われるような提案をしてしまった。

「お前……正気か？」
「正気じゃないかもです。でも、私達のアカウントなんて誰も気に留めてませんよ」
「いやいや。それでも、もし見られたら超恥ずかしいぞ」
「でも、せっかく恋人になったからのろけてみたいんです」
「全世界に向けてノロける必要あるか？」
「それは無いですけど……もちろん、無理にとは言わないですが」
「はあ……まあいいか。じゃあ、それなら実名はなしな。あだ名はいいけど」
「さすがにそこがバレるとまずいですしね。了解です！」
「りんが露出狂だって初めて知ったよ」
「露出狂じゃないです。惚気けたいだけなんです！」
「はいはい。もうなんでもいいよ」

　先輩がなんか投げやりになって来た。
　見られなくてもいいし、見られたらそれはそれで面白い反応が来そう。
　そんな事を思っている私はやっぱり変なのだろう。
　でも、それでも幸せだしいいや。

というわけで、交換日記の日々が始まります。
「日記」なので出来れば毎日更新で、とりあえず一週間は更新継続予定です。
気力が尽きたら終了で、続けられそうなら一ヶ月くらいはやってみようかなと。
GitHub交換日記を始めたアホな二人の日常をお楽しみいただければ。
